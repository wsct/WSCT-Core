Namespace WSCT.Helpers.BasicEncodingRules

Classes

AbstractTlvObject

Base abstract class of all TLV data interpretation. It allows to bind a TlvData object with the associated TlvDescription object describing it.

BcdTlvObject

Represents a TLV object where the value is BCD encoded.

BinaryTlvObject

Represents a TLV object where the value is binary data.

StringTlvObject

Represents a TLV object where the value is a string

TlvData

Represents data formatted in BER TLV format.

TlvDataHelper

Allows simple manipulation and conversion between byte[], string and TlvData types data.

ArrayOfBytes also provides extension methods on byte[] and string and List<TLVData>

TlvDescription

Describes the TLV representation of a given tag.

TlvDescription.ValueType

Description of field value.

TlvDictionary

Manages a list of known TLV data representations.

List can be programmatically defined of read from an XML file (see Dictionary.EMVTag.xsd for XML format).
