Class TlvDictionary

Manages a list of known TLV data representations.

List can be programmatically defined of read from an XML file (see Dictionary.EMVTag.xsd for XML format).

Remarks

AbstractTlvObject is the base abstract class for all TLV data representations. An instance of TlvDescription explains how to create a TLV data representation for a given tag. TlvDictionary allows to manage sets of TlvDescription.
