Examples
TLVData = new TLVData("6F 1A 84 0E 31 50 41 59 2E 53 59 53 2E 44 44 46 30 31 A5 08 88 01 02 5F 2D 02 66 72");

Default format: 

String.Format("{0}", tlv);
output: T:6F L:1A V:( T:84 L:0E V:31 50 41 59 2E 53 59 53 2E 44 44 46 30 31 )( T:A5 L:08 V:( T:88 L:01 V:02 )( T:5F2D L:02 V:66 72 ) )

Format "T": tag only 

String.Format("{0:T}", tlv);
output: 6F

Format "L": length only 

String.Format("{0:L}", tlv);
output: 1A

Format "V": value only; value is interpreted as TLV if its a complex tag 

String.Format("{0:V}", tlv);
output: ( T:84 L:0E V:31 50 41 59 2E 53 59 53 2E 44 44 46 30 31 )( T:A5 L:08 V:( T:88 L:01 V:02 )( T:5F2D L:02 V:66 72 ) )

Format "Vh": raw value, no interpretation 

String.Format("{0:Vh}", tlv);
output: 84 0E 31 50 41 59 2E 53 59 53 2E 44 44 46 30 31 A5 08 88 01 02 5F 2D 02 66 72
